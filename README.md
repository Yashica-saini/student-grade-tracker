# Student Grade Tracker

## Overview

This project is a **command-line Student Grade Tracker** built using Python.
The purpose of this project is to practice working with Python data structures and basic programming logic.

The system allows users to store student information, calculate results, and analyze grade data using simple Python features.
---
## Features

### Add Student Records

The program allows adding students with their marks for different subjects.

### Store Data Using Dictionaries

Student data is stored using Python dictionaries, which makes it easy to organize and retrieve information.

### Calculate Results

The system calculates:

* Total marks
* Average marks
* Student grades based on performance

### Display Student Data

Users can view all stored student records along with their calculated results.

### Basic Data Analysis

The program can analyze student performance and display useful information such as:

* Highest scoring student
* Average class performance
* Individual student results

## Technologies Used

* Python 3

* Loops and Conditional Statements

## How to Run the Program

1. Make sure Python is installed on your computer.
2. Clone or download this repository.
3. Run the Python file using the terminal or command prompt.

Example:

```
python student_grade_tracker.py
```
